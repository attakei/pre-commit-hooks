My pre-commit hooks collection
==============================

Usage
-----

Add using hooks into your ``.pre-commit-config.yaml``.

.. code-block::

   repos:

     - repo: https://github.com/attakei/pre-commit-hooks
       rev: v0.0.0
       hooks:
         - id: conventional-commits
